# Meu Primeiro Site

Este é um projeto simples de página web criado para praticar conceitos básicos de desenvolvimento frontend.

## Tecnologias utilizadas
- HTML
- CSS

## Funcionalidades
- Página simples de apresentação
- Estrutura básica de um site
- Estilização com CSS

## Objetivo
Praticar a criação de páginas web e começar a construir um portfólio de projetos no GitHub.

## Como visualizar
1. Baixe ou clone o repositório
2. Abra o arquivo index.html no navegador
